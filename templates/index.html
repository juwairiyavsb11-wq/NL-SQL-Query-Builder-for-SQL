    Help — AskDB

    Getting Started

    Upload a SQLite database file (.db, .sqlite, .sqlite3) to begin exploring your data using natural language.

    Example Queries

    • Show all customers
    • Top 10 products by price
    • How many orders were placed this month?
    • Average order total by status
    • Customers from New York who spent more than 100
    • Count products by category

    Supported Operations

    AskDB supports read-only queries: SELECT, GROUP BY, ORDER BY, LIMIT, HAVING, WHERE, and JOINs.

    For safety, operations like DELETE, DROP, UPDATE, INSERT are blocked.

    Export

    Export your results as CSV or Excel using the buttons above the results table.

    Keyboard Shortcuts

    • Ctrl + Enter — Submit query
    • Escape — Close modals
